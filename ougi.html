Ougi Bot

Does spooky stuff.

A simple chat bot with an undefined personality: If you had to describe it on the shortest phrase, it would be "Oshino Ougi is Oshino Ougi". It's designed to respond messages, send memes, do maths (kinda, still studying), say your nonsense, answer questions (boolean-type responses for now) and improving almost everyday! This project is a hobby I enjoy developing on my free time, (even though I doubt anyone would be looking forward for progress), please be patient. As of now, get started by asking Ougi for help, using "ougi help", I'm sure it'll gladly answer.
